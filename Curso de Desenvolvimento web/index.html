Curso de Front-End

Aprenda HTML, CSS e JavaScript criando games do zero

Como o curso ira funcionar:

Nos dias de hoje, com a popularização das IAs (inteligências artificiais), a programação pode parecer mais simples, especialmente quando essas ferramentas são utilizadas como apoio no processo de desenvolvimento. Muitos programadores experientes fazem uso da IA para otimizar seu trabalho, seja para automatizar tarefas repetitivas, revisar código ou até mesmo gerar soluções para problemas específicos.

No entanto, para quem está começando na área, o uso desenfreado dessas ferramentas pode criar uma falsa sensação de aprendizado. Ao resolver exercícios ou problemas de programação com a ajuda de IAs como o Chat GPT, alguns iniciantes acreditam que estão dominando os conceitos, quando, na verdade, estão apenas copiando respostas sem compreender a lógica por trás delas.

Essa situação gera um efeito cascata: cada vez mais pessoas negligenciam os fundamentos da programação, o que resulta no desenvolvimento de softwares frágeis, mal estruturados e mais propensos a erros.

Pensando nisso, o IFRJ (Instituto Federal do Rio de Janeiro) – Campus Engenheiro Paulo de Frontin, em parceria com o LabProjNTE e a BitNinjas elaborou um curso de Desenvolvimento Web, com carga horária total de 40 horas e modalidade híbrida – com aulas online e presenciais.

Nosso objetivo é abordar conceitos-chave do desenvolvimento web, permitindo que os alunos construam uma base sólida em programação, com foco em práticas atuais e aplicáveis no mercado.

Modulos do curso:

  1. Marcando com HTML ➔ Introdução à linguagem de marcação responsável por estruturar o conteúdo das páginas web.
  2. Estilizando com CSS ➔ Aprendizado sobre como aplicar estilos visuais, cores, layouts e animações às páginas.
  3. Programando com JavaScript ➔ Exploração da linguagem de programação que dá interatividade e dinamismo aos sites.
  4. Desenvolvendo Mini-Games Web ➔ Aplicação prática dos conhecimentos adquiridos na criação de pequenos jogos interativos para navegador.

Com essa estrutura, o curso aborda as três principais linguagens da Web: HTML, CSS e JavaScript. Oferecendo assim, uma base sólida para quem deseja iniciar na área de desenvolvimento front-end.

Como funcionarão as aulas:

As aulas do curso serão modalidade híbrida – com aulas online e presenciais. As aulas presenciais serão no campus do IFRJ – Engenheiro Paulo de Frontin, no período da manhã.

As aulas serão ministradas pelo professor Leonardo S. Souza.

Em caso de dúvidas, procure o docente Roni Costa.
